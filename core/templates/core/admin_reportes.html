Leyenda de formatos de descarga:

CSV: Entrega todos los datos en una sola fila (EXCEL), separado por comas sin ningun ordenamiento. No se recomienda utilizar para revisión de datos, informes y/o resumenes.

XLS y XLSX: Entrega los datos ordenados segun su tabla e Identificador unico (EXCEL). Se recomienda para obtener datos y dar una revision general de la tabla.

TSV: Entrega los datos en un bloc de notas ordenados segun Identificador Unico

ODS: No entrega ningun resultado visible por Excel, por lo cual no se debe utilizar.

JSON: Entrega los datos ordenados para se utilizados bajo JavaScript, este informe esta orientado hacia los desarrolladores de la pagina, por lo cual no se recomienda utilizar para revisión de datos.

YAML: Entrega los datos bajo el siguiente formato: "Nombre de tabla" : "Identificador Unico". Ej: {Empleado : 1}

HTML: Entrega los datos bajo formato JSON, y los muestra en una nueva pagina del navegador.
